# **Imani's Developer Portfolio**

Welcome to my personal developer portfolio built using **Flask** and **Tailwind CSS**.

This portfolio showcases my skills in **Web Development**, **Cybersecurity**, and more, with a contact form powered by Flask's built-in server-side handling.

## **Features**

- Personal profile with an introduction and background.
- Contact form to send messages.
- Custom 404 page when a user navigates to a non-existing page.
- Beautiful design with **Tailwind CSS**.
- Deployment on **Render** for fast hosting with automatic HTTPS support.

## **Tech Stack**

- **Backend:** Python with Flask
- **Frontend:** HTML, CSS (Tailwind CSS)
- **Hosting:** Render
- **SMTP:** /*i am not done with this*/ Email integration using Gmail for sending contact form messages.
